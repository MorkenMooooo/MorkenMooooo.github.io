产品中心 ​

门主引擎致力于为开发者提供开箱即用的嵌入式AI智能硬件解决方案。

🔧 硬件产品 ​

Sovereign DevKit v1.0 ​

一款专为边缘AI推理设计的开发板。

  • 主控: ESP32-S3 (双核, 512KB SRAM)
  • AI加速: 内置向量指令集
  • 传感器: 集成麦克风、6轴IMU
  • 特色: 板载 USB-JTAG 调试器，无需额外调试器即可进行深度调试。
  • 状态: 开发中，预计 Q3 2026 开放预订。

🛠 软件工具 ​

Sovereign CLI ​

一个命令行工具，用于简化模型转换、设备烧录和日志监控。

  • 功能: idf.py flash, idf.py monitor, idf.py build
  • 开源地址: GitHub - MorkenMooooo/SovereignEngine
